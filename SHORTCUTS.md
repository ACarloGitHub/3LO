# Scorciatoie da tastiera 3LO

## Nella Board

| Tasto | Azione |
|-------|--------|
| Ctrl + + | Zoom in |
| Ctrl + - | Zoom out |
| Ctrl + 0 | Reset zoom |
| Invio (textarea) | Crea scheda |
| Maiusc + Invio (textarea) | A capo nella scheda |
| Trascina board | Scroll orizzontale |
| Trascina vicino bordi | Auto-scroll accelerato |

## Generale

| Tasto | Azione |
|-------|--------|
| Ctrl + Q | Chiudi app |
