Tasti Scorciatoia

  • Ctrl + Scroll → Zoom in/out board
  • Ctrl + +/- → Zoom pulsanti
  • Invio → Crea nuova scheda
  • Maiusc + Invio → A capo nella scheda
  • Doppio click → Modifica titolo progetto
  • Drag & Drop → Sposta schede tra liste
